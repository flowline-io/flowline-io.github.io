capability:kanboard.complete_task

Complete a task (mutation)

Inputs (params):

Param Type Required Description
id int yes Task ID

Outputs: InvokeResult JSON (see ../capabilities.md). Read domain fields under data; use text when present.

Usage:

  - id: complete_task_step
    action: capability:kanboard.complete_task
    params:
      id: 0  # required
